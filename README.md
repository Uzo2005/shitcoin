#### How it works
Nodes can sync and broadcast. Clients can make and broadcast transactions. Mining increases nonce until sha256 hash starts with 4 zeroes.
Nodes verify transaction signature before processing it.

To run it:
 1. Install Nim
 2. Clone the repo
 3. Run `./setup.sh`
 4. To run a node
    ```sh
        cd node/
        nim r node.nim
    ```
 5. To run a client
    ```sh
        cd client/
        nim r client.nim
    ```
